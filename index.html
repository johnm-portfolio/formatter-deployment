Installation

Get started in seconds. Install the package directly from GitHub:

pip install git+https://github.com/johnm-portfolio/formatter-deployment

Or clone and install locally:

git clone https://github.com/johnm-portfolio/formatter-deployment.git note_formatter
cd note_formatter
pip install .

Using the Package

Interactive mode:
note-formatter
Format and save in place:
note-formatter "file.md" --inplace
Format with custom output:
note-formatter "input.md" -o "output.md"
Get help:
note-formatter --help

Windows Shortcut

For even faster access on Windows, create a shortcut that launches the formatter:

The GitHub repo has a run_cli.lnk shortcut already

  1. Right-click on your desktop → New → Shortcut
  2. Enter the location:
    powershell.exe -Command "note-formatter --inplace"
  3. Or replace the note-formatter --inplace with a different CLI call

Now you can quickly format files by clicking the shortcut, and can pin it to your taskbar for efficiency.

How It Works

This tool converts custom symbol representations into proper text symbols and processed formats. It's designed for efficient note-taking during lectures where finding and copying symbols would be distracting.

✓
Character Sterilization: Replaces custom quotation marks and special characters with proper Unicode equivalents
✓
Auto TOC Generation: Structures your notes with automatically generated table of contents
✓
Intuitive Mappings: Common representations are easy to remember with multiple aliases for convenience

Why Build This?

Taking notes in university lectures requires focus and direct attention. This tool eliminates the hassle of finding and copying symbols, letting you type intuitive shorthand representations during lectures and format them afterwards.

This tool can, of course, be used for general note taking as it provides an automatic Table of Contents generation and unicode cleanup.

With regular use, common representations become second nature. But if you forget, the interactive symbol table is always available for quick reference.
